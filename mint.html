Get your Node License

Follow these steps to acquire a BCCS Infrastructure Access License. Works on mobile and desktop.

Get a Crypto Wallet

You need a crypto wallet. If you don't have one, download one of these:

Coinbase Wallet Easiest for beginners MetaMask Most popular

What is a wallet? A digital account you fully control. Nobody else has access. It stores your digital assets and lets you interact with blockchain apps.

Get USDC on Base

You need $1,000 USDC on the Base network, plus a tiny amount of ETH (under $0.10) for fees.

In Coinbase Wallet: tap "Buy" → select USDC → choose Base → pay with card.

From Coinbase exchange: send USDC to your wallet → select Base as network.

From Ethereum: bridge via bridge.base.org

You're a Node Operator

Your soulbound Node License is in your wallet. It cannot be transferred or sold.

Next: When the verification network launches, you'll operate a node, process verification requests, and earn $BCCS tokens for honest work.

Common Questions

Never used crypto before?
Start with Coinbase Wallet — buy USDC with a bank card. Zero to Node License in ~15 minutes.
Is this an investment?
No. A Node License is a software permit. You earn tokens through work, not by holding. No guaranteed return.
Can I sell my license?
No. Soulbound = non-transferable. Prevents speculation, ensures committed operators.
Do I need to be whitelisted?
Yes. Apply on the BCCS homepage first.
What does a node do?
Processes biological asset verification requests using satellite, sensor, and inspection data. Honest work earns $BCCS. Dishonest submissions are penalized.

A BCCS Infrastructure Access License is a software permit, not an investment product. It does not include any verification unit allocation. Validator emissions begin only when the mainnet verification network is operational. KRYONIS Sovereign Systems Limited, Hong Kong SAR.
