ACQUIRE NODE LICENSE

Get your Node License

Follow these steps to acquire a BCCS Infrastructure Access License. The entire process takes about 10 minutes if you already have a crypto wallet.

Get a Crypto Wallet

You need a crypto wallet to interact with the blockchain. If you don't have one, we recommend Coinbase Wallet (easiest for beginners) or MetaMask (most popular).

What is a wallet? It's like a digital bank account that you control. Nobody else has access — only you. It stores your digital assets and lets you interact with blockchain applications.

Already have a wallet? Make sure it supports the Base network. Coinbase Wallet, MetaMask, and Rainbow all work. You can skip to Step 2.

Download Coinbase Wallet
Switch to Base Network

Base is a Layer 2 blockchain built by Coinbase. It's faster and cheaper than Ethereum mainnet. All BCCS contracts live on Base.

How to switch:

In Coinbase Wallet: tap the network selector at the top → search "Base" → select it.

In MetaMask: tap the network dropdown → "Add network" → search "Base" → add and switch. Or visit chainlist.org/chain/8453 and click "Add to MetaMask".

Base network details: Chain ID: 8453 · RPC: https://mainnet.base.org · Explorer: basescan.org

Get USDC on Base

Node Licenses are paid in USDC — a stablecoin pegged to the US Dollar. 1 USDC = $1. You need $1,000 USDC for an Alpha tier license, plus a tiny amount of ETH (under $0.10) for transaction fees.

Easiest way — buy directly:

In Coinbase Wallet: tap "Buy" → select USDC → choose Base network → pay with bank/card. The USDC arrives directly in your wallet on Base.

Already have USDC on Ethereum? Bridge it to Base using the official Base Bridge. This takes a few minutes.

Already have USDC on a Coinbase account? Send it to your Coinbase Wallet address. Make sure you select Base as the network when sending.

What happens when you mint: Your wallet will ask you to approve two transactions. First, you approve the USDC spend (this gives the contract permission to take $1,000 USDC). Second, you confirm the mint (this creates your soulbound Node License NFT and sends the USDC to the protocol treasury). Once confirmed, your license appears in your wallet as an NFT.

You're a Node Operator

Your soulbound Node License is now in your wallet. It cannot be transferred or sold — it is permanently bound to your address.

What happens next: When the BCCS verification network launches, you'll be able to operate a verification node using this license. You'll process biological asset verification requests, submit evidence data, and earn $BCCS tokens for honest work.

Until then, your license confirms your position as an early infrastructure operator in the BCCS network.

Common Questions

What if I've never used crypto before?
Start with Coinbase Wallet — it's designed for beginners. You can buy USDC directly with a bank card. The whole process from zero to Node License takes about 15 minutes.
Is this an investment?
No. A Node License is a software permit that gives you the right to operate verification infrastructure. You earn tokens through work, not by holding a license. There is no guaranteed return.
Can I sell my license later?
No. Licenses are soulbound (non-transferable). This is by design — it ensures every operator is committed, prevents speculation, and maintains network integrity.
Why USDC instead of ETH or credit card?
USDC is a regulated stablecoin worth $1. It eliminates price volatility — you always know exactly what you're paying. The protocol treasury receives a known dollar amount, enabling predictable infrastructure funding.
What is Base?
Base is a Layer 2 blockchain built by Coinbase. It's secured by Ethereum but transactions are faster and much cheaper (fractions of a cent). All BCCS contracts are deployed on Base.
Do I need to be whitelisted?
Yes. Alpha tier requires whitelist approval. Submit your application on the BCCS homepage and you'll be reviewed for inclusion.
What does a verification node actually do?
When operational, nodes process biological asset verification requests. You receive data from satellites, sensors, and inspectors — you validate that data against protocol rules — and you submit verification confirmations on-chain. Honest work earns $BCCS tokens. Dishonest submissions are penalized (slashed).

A BCCS Infrastructure Access License is a software permit, not an investment product. It does not include any verification unit allocation. Validator emissions begin only when the mainnet verification network is operational. No representation is made regarding the future value of any digital asset associated with this protocol.
